## Что такое бот Энигма?
[Enigma — это Telegram-бот](https://t.me/UZBEKGOATSBOT?start=C96FP5YS), который выполняет поиск данных в утёкших базах и открытых источниках. Он сопоставляет номер телефона, email, ФИО или никнейм с доступными записями и формирует структурированный отчёт.

Бот работает не через взлом, а через поиск в:
* Базах данных, оказавшихся в открытом доступе
* Утечках и слитых каталогах
* OSINT-источниках и публичных сервисах

В отличие от Funstat (который анализирует активность внутри Telegram), Enigma занимается именно пробивом по базам и идентификаторам.

## Что умеет бот Энигма?
Функционал Enigma сосредоточен на пробиве данных по ключевым параметрам. Основные возможности:

* Поиск по номеру телефона — выявление возможных привязок к сервисам и утечкам.
* Поиск по email — проверка попадания почты в базы данных и сопоставление с аккаунтами.
* Поиск по username / нику / Telegram ID — привязки к другим данным, если они присутствуют в утечках.
* Поиск по ФИО — пересечение с публичными каталогами и документами.
* Формирование отчётов — бот собирает всё найденное в структурированный результат.

Таким образом, бот Энигма — инструмент именно поиска в базах данных.

## Для чего используют Телеграм бот Enigma?
Бот Enigma применяют в следующих ситуациях:

* OSINT-расследования — поиск цифрового следа по человеку.
* Проверка личности — бывает полезно при знакомстве, найме, проверке контрагентов.
* Кибербезопасность — поиск утечек персональной информации.
* Поиск информации «для себя» — проверка собственных данных на попадание в базы.

У Enigma есть бесплатные функции, но для глубокого пробива обычно требуется пополнение баланса или участие в реферальной программе.

## Советы по использованию бота Энигма
Чтобы получать максимальную пользу:
* Начинайте с простых запросов: телефон → email → ФИО → username.
* Сравнивайте результаты с другими ботами — это повышает точность данных.
* Не делитесь конфиденциальными данными: Telegram ID и username достаточно.
* Проверяйте актуальную ссылку Enigma — не используйте сомнительных клонов.

Бот не предназначен для незаконных действий.

Пользователь несёт ответственность за законность своих запросов.
